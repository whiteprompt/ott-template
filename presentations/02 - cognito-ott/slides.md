# AWS Cognito

### What is Cognito?

- AWS Cognito is a simple user identity service that helps you securely manage and synchronize app data for your users across their devices.

<br><br>

### Benefits of using Cognito?

- Easy configuration. 
- Triggers
- Securely store sensitive information of the user
- Amplify library can fit several use cases

<br><br>

### Caveats
- Pool settings should be set at creation time.
